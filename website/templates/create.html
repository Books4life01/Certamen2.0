LIVE CERTAMEN

Hosting a LIVE Game allows multiple players to join indivisual rooms and participate in a virtual live certamen game. Host send out questions from their computers and contestants buzz in to answer.

SCOREBOARD CERTAMEN

A SCOREBOARD Game simulates a virtual Certamen tournament scoring system. Host can create rooms where other oher hosts(to whom the room key has been shared) can virtually score the results of a certamen game played in person. These results will be compiled providing quick and detailed game and tournamen reports.
